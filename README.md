# Email Spam Classifier

Machine Learning and NLP-based web application that classifies messages as Spam or Ham.

## Live Demo
https://saniya3540-email-spam-classifier-app-q5rc74.streamlit.app/

## Features
- Text preprocessing using NLP
- Stopword removal
- Stemming using PorterStemmer
- TF-IDF Vectorization
- Spam/Ham prediction
- Model saving using Pickle
- Streamlit deployment

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Streamlit

## NLP Techniques Used
- Text Cleaning
- Stopword Removal
- Stemming
- TF-IDF Vectorization

## Machine Learning Algorithm
- Logistic Regression

## Model Performance
- Accuracy: 97.84%
